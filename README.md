# Hybridized-Phi4-Criticality-Optimizer
A percolation-physics rooted optimization tool for the Phi4 Model, succesfully recovering the industry-standard values (https://arxiv.org/pdf/2512.16536). The model is based of the logic of the Invaded Cluster Algorithm (Machta, 1995), leveraging the notion that a system is critical when percolation appears, but adapted to become an optimization tool. The advantage of this model is its topological awareness, and its homing towards criticality with each Monte Carlo step; the model also provides a significant speedup as it measures the critical components natively rather than creating an ensemble of traditional measurements. 

The framework provided here can easily be extended to other models; however, the code provided is strictly formulated or the Phi4 model, and is not production optimized.
